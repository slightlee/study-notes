### netstat -tulnp
```shell
netstat -tulnp

-t 表示显示 TCP 端口，
-u 表示显示 UDP 端口，
-l 表示显示监听状态的端口，
-n 表示以数字形式显示地址和端口号，
-p 表示显示与进程相关的信息。
```

### lsof -i -P -n
```shell
lsof -i -P -n

这会列出所有网络连接以及每个连接所对应的进程和程序。 
-i 表示仅显示网络相关的信息，
-P 表示不将端口号转换为服务名，
-n 表示以数字形式显示地址和端口号。

```
